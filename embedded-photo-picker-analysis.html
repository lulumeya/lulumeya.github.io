Google Messages가 Embedded Photo Picker를 정식 도입했습니다 (2026.01)

Google Google Messages

카메라 버튼 아래에 Photo Picker가 인라인으로 배치되어, 사진 촬영과 갤러리 선택을 한 화면에서 처리합니다. 스크롤하면 전체 화면으로 확장되며 Google Photos 클라우드 콘텐츠도 탐색 가능합니다.

Before / After 기존 Photo Picker vs Embedded Photo Picker

Google Messages에서 기존 방식과 Embedded 방식의 실제 동작 차이입니다. 기존에는 전체 화면이 전환되지만, Embedded는 앱 안에서 자연스럽게 사진을 탐색합니다.

AFTER: Embedded Photo Picker

앱 내 인라인 선택

Google이 직접 선도: Google Messages가 가장 먼저 Embedded Photo Picker를 도입했으며, Android 공식 블로그에서 모든 앱에 도입을 권장하고 있습니다. 사진 선택이 핵심인 우리 앱에서도 동일한 UX 개선 효과를 기대할 수 있습니다.

Customization

디자이너가 커스터마이즈할 수 있는 영역

Embedded Photo Picker는 시스템 UI이지만, 아래 항목들은 앱에 맞게 조정할 수 있습니다

🎨 Accent Color 커스텀 가능

선택된 사진의 체크마크, 선택 카운트 뱃지, 하이라이트 등 Picker 전체의 포인트 컬러를 앱 브랜드 컬러에 맞출 수 있습니다.

Long 형식 ARGB 컬러
🌓 Dark / Light Theme 커스텀 가능

앱의 테마 모드에 맞춰 다크 모드 / 라이트 모드 / 시스템 설정 따르기 중 선택 가능합니다. 앱 전체 테마와 자연스럽게 통일할 수 있습니다.

Light Dark System
#️⃣ 최대 선택 개수 커스텀 가능

사용자가 선택할 수 있는 사진/영상의 최대 개수를 지정합니다. 화면별로 "프로필은 1장", "후기는 최대 10장" 등 유즈케이스에 맞게 제한 가능합니다.

📁 파일 타입 필터 커스텀 가능

사진만 / 영상만 / GIF만 / 특정 포맷만 보여줄 수 있습니다. MIME 타입으로 필터링하여 유저에게 불필요한 파일을 숨길 수 있습니다.

image/* video/* image/gif
🔢 순서 선택 모드 커스텀 가능

활성화하면 선택한 순서대로 1, 2, 3... 번호가 뱃지로 표시됩니다. 후기 작성처럼 사진 순서가 중요한 UX에서 유용합니다.

📐 크기 / 배치 영역 커스텀 가능

Picker를 앱 레이아웃 안 원하는 위치와 크기로 임베드합니다. 하단 시트, 반절 화면, 전체 화면 등 자유롭게 배치할 수 있습니다.

커스터마이즈 불가 영역 (시스템 제공)
  • 그리드 레이아웃 — 사진 격자의 열 수, 간격, 썸네일 비율은 시스템이 결정
  • 내부 네비게이션 — 앨범 목록, 검색 UI, 탭 구조 등은 OS가 제공하는 그대로
  • 선택 애니메이션 — 체크마크 등장, 확대/축소 전환 등 인터랙션
  • 폰트 / 아이콘 — Picker 내부의 텍스트 스타일, 아이콘은 변경 불가
디자이너를 위한 요약: Embedded Photo Picker는 "시스템이 제공하는 갤러리 뷰"를 앱 안에 끼워 넣는 방식입니다. 브랜드 컬러, 테마, 크기, 위치, 선택 규칙은 맞출 수 있지만, 내부 UI 디테일(그리드, 폰트, 아이콘)은 OS 통일 경험을 위해 변경할 수 없습니다. Google Messages처럼 하단 시트에 자연스럽게 배치하는 것이 가장 검증된 패턴입니다.

제안 요약

  • Embedded Photo Picker는 앱 이탈 없는 자연스러운 사진 선택 UX를 제공합니다.
  • 우리 Android 유저의 89%가 이미 지원 대상이고, 이 비중은 계속 증가하고 있습니다.
  • 미지원 유저에게는 기존 Picker로 fallback하면 되므로 도입 리스크가 낮습니다.
  • Google 공식 API로 장기적 유지보수와 호환성이 보장됩니다.
